외부 스타일 시트

id 선택자

id 선택자는 #id명으로 지정

class 선택자

class 선택자는 .클래스명으로 선택

클래스는 중첩해서 여러개를 같이 쓸 수 있다.

다중 클래스를 선택할 때는 .으로 연결해서 선택한다.
